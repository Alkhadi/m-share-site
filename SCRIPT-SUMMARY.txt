================================================================================
  ✨ WHAT THE SCRIPT DOES
================================================================================

✅ Extracts header & footer from index.html
✅ Replaces them on all HTML pages site-wide
✅ Adds ChatGPT 5 Pro typography (gpt5-typography class)
✅ Creates readability CSS with professional contrast
✅ Ensures footer has Explorer section with all navigation
✅ Ensures footer has Theme controls for color changes
✅ Creates timestamped backups (*.oneshot.bak.YYYY-MM-DDTHH-MM-SSZ)
✅ Idempotent - safe to run multiple times

  🆘 TROUBLESHOOTING
================================================================================

Problem: "ERROR: index.html not found"
Solution: Run from site root directory

Problem: Changes not visible
Solution: Clear browser cache (Ctrl+Shift+R)

Problem: Theme controls not working
Solution: Check browser console, clear localStorage

Problem: Footer not showing
Solution: Verify index.html has <footer> element
