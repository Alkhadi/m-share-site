Time‑Delay Prompting

What: Give the cue, wait briefly, then prompt only if needed. Increase the wait over time.

  1. Present clear cue (e.g., “Read name”).
  2. Wait 1–3 s; prompt if no response.
  3. Reinforce the independent response.
  4. Lengthen delay across days.
