High‑Probability Request Sequence

What: Give 2–3 easy “yes” tasks to build momentum, then the target instruction.

  1. Pick 2–3 certain‑success requests.
  2. Reinforce quick compliance.
  3. Deliver target request immediately after.
  4. Fade over time.
