One‑Page Profiles & Passports

What: A one‑page summary of strengths, communication, and adjustments so all adults support consistently.

  1. Co‑produce with pupil/family; include “What people like about me”, “How I communicate”, “Best support”.
  2. Use clear icons/pictures; keep to one page.
  3. Share with all staff and update each term.

References

  • DfE SEND Code of Practice (reasonable adjustments; graduated response).
  • Council for Disabled Children — one‑page profile resources.
