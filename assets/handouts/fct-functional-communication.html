Functional Communication Training (FCT)

What: Teach a simple way to ask (card/word/device) that achieves the same outcome as the behaviour.

How: Identify the function (get/escape/attention/sensory) via brief ABC, then prompt the new request and reinforce it immediately.

Steps

  1. Define one behaviour and its function.
  2. Choose the replacement (e.g., “help”, “break”, “more”).
  3. Prompt → reinforce → fade prompts.
  4. Train all staff for consistency.
